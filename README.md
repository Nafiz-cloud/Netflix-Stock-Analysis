📊 Netflix Stock Analysis (2003 - 2020)
An Exploratory Data Analysis (EDA) of Netflix (NFLX) stock history using Python to identify growth trends and market volatility.

🚀 Features
Time-Series Analysis: Visualized stock price evolution over 17 years.

Volume Tracking: Analyzed trading volume vs. time to measure market interest.

Extreme Values: Identified the Top 5 High and Top 5 Low price points.

Temporal Trends: Analyzed price fluctuations by day, month, and year.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, Matplotlib, Seaborn

Tools: Jupyter Notebook

📈 Key Insight
The project highlights Netflix's transition from a niche service to a global market leader, tracking its growth from under $2 to over $500.
